MSE vs inner-product quantization

The asymmetry

There are two natural quality metrics for a quantizer $Q$ on vectors:

  • MSE / reconstruction error: $\mathbb{E}\|x - Q(x)\|_2^2$
  • Inner-product / search error: $\mathbb{E}(\langle y, Q(x)\rangle - \langle y, x\rangle)^2$ for a held-out query $y$

Most classical quantization theory targets MSE — Lloyd-Max, Shannon’s rate-distortion bound, vector quantization, Linde-Buzo-Gray, all minimize $\ell_2$ reconstruction. But nearest-neighbor search and attention both need inner products to be preserved, not raw vectors. The two objectives diverge in a specific, predictable way.

Why MSE-optimal quantizers are biased for inner products

Consider 1-bit MSE-optimal quantization of a coordinate $x \sim \mathcal{N}(0, 1/d)$: the optimal codepoints are $\pm \mathbb{E}|x| = \pm\sqrt{2/(\pi d)}$. So

$$Q_{\text{mse}}(x) = \sqrt{\tfrac{2}{\pi d}} \cdot \text{sign}(x)$$

For a query $y$ in expectation:

$$\mathbb{E}_x \langle y, Q_{\text{mse}}(x)\rangle = \sqrt{\tfrac{2}{\pi d}} \mathbb{E}\langle y, \text{sign}(x)\rangle = \tfrac{2}{\pi} \langle y, x\rangle$$

So the MSE-optimal quantizer underestimates inner products by a factor of $2/\pi \approx 0.64$. This is structural, not a bug — minimizing MSE pulls the codepoints inward (to where the mass is); preserving inner products needs codepoints further out.

You can rescale to fix the multiplicative bias (multiply by $\pi/2$), but only at the cost of inflating variance. There’s a real Pareto frontier between MSE and inner-product accuracy.

Three fixes, three modern algorithms

1. Ratio normalization (RaBitQ)

Estimate $\langle y, x\rangle$ as $\langle y, \tilde{x}\rangle / \langle \tilde{x}, x\rangle$ where $\tilde{x}$ is the quantized vector and $\langle \tilde{x}, x\rangle$ is a single scalar stored per vector. The ratio is unbiased without rescaling. Costs: $O(1)$ extra storage per vector. See RaBitQ (Gao et al., 2024).

2. Asymmetric estimator (QJL)

Quantize one side only. Apply sign-bit to the stored vector $x$, keep the query $y$ in full precision. The estimator

$$\hat{\langle y, x\rangle} = \frac{\pi/2}{m} \|x\|_2 (Sy)^T \text{sign}(Sx)$$

is unbiased — the $\pi/2$ rescaling works here because variance is bounded by the JL embedding. See QJL (Zandieh et al., 2024).

3. Two-stage MSE + 1-bit residual (TurboQuant)

Quantize the bulk of the signal with an MSE-optimal Lloyd-Max scheme (accepts the inner-product bias), then estimate the residual with an unbiased 1-bit QJL. The residual has small norm, so the unbiased estimator’s variance is small. Combined estimator is unbiased and inherits near-Shannon-rate MSE on the first stage. See TurboQuant (Zandieh et al., 2025).

Pointer to think about it

A useful way to see why these aren’t all the same:

Method MSE near-optimal? Inner-product unbiased?
Lloyd-Max (raw) Yes No (factor-$2/\pi$ bias)
Lloyd-Max + $\pi/2$ rescale Inflated variance Yes
RaBitQ ratio Yes (after rescale) Yes
QJL N/A (lossy projection) Yes
TurboQuant 2-stage Yes Yes

TurboQuant is the first construction to be near-optimal on both axes simultaneously across arbitrary bit-widths.
