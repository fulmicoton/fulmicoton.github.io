Titans (Behrouz et al., 2024)

researchSource PDF: behrouz-2024-titans.pdf
Part of
Syllabus — Memory for AI agents

Summary

Titans (Behrouz, Zhong, Mirrokni — Google, 2024) introduces a neural long-term memory module that learns to memorize at test time — its weights update during inference based on a “surprise” signal, with momentum and forgetting gates inspired by classical optimizer dynamics. The premise: attention is excellent short-term memory but bounded by context; explicit RNN/SSM state is long-term but lossy. Titans combines a frozen attention “core” with a parametric memory module updated online by gradient descent on a surprise loss (how poorly the module predicts the current key→value). Three integration variants — Memory-as-Context (MAC), Memory-as-Gate (MAG), Memory-as-Layer (MAL) — explore how the memory should interact with the core attention. Evaluated on language modeling, commonsense reasoning, genomics, and time series with context lengths beyond 2M tokens, Titans outperform Transformers and modern linear recurrent baselines (Mamba, RWKV, DeltaNet) on long-context retrieval (needle-in-haystack), and the test-time-learned memory is the direct conceptual ancestor of δ-mem.

Key points

Memory as test-time learning

The core idea reframes “memory” as an inner optimization loop. At each step, the memory module $\mathcal{M}_t$ (a small MLP) is updated by gradient descent on a loss measuring how well $\mathcal{M}_{t-1}$ predicts the value given the key at the current position:

$$\ell_t(\mathcal{M}) = \|\mathcal{M}(k_t) - v_t\|^2$$

The update follows a momentum + decay rule:

$$\mathcal{M}_t \leftarrow (1 - \alpha_t) \mathcal{M}_{t-1} + \eta_t \cdot m_t, \quad m_t = \beta_t m_{t-1} - \theta_t \nabla \ell_t(\mathcal{M}_{t-1})$$

with all gates ($\alpha_t, \eta_t, \beta_t, \theta_t$) generated from the current token. Surprise = the gradient norm; high-surprise tokens get written more strongly. The forget gate $\alpha_t$ enables graded forgetting.

Three integration variants

The paper sweeps how the test-time-learned memory module sits relative to the attention core:

  • Memory-as-Context (MAC) — memory output is prepended to the attention context as additional persistent tokens. Attention treats memory as a fixed-length extra window.
  • Memory-as-Gate (MAG) — memory output and attention output are mixed via a learned gate (sliding-window attention for short-term, memory for long-term).
  • Memory-as-Layer (MAL) — memory is a layer that transforms the sequence before attention sees it; the attention core then operates on the memory-augmented stream.

MAG is the best on language modeling; MAC is the best on long-context retrieval (needle-in-haystack); MAL is intermediate.

Persistent memory tokens

Beyond the test-time-learned dynamic memory, Titans also have a small set of persistent memory tokens — learnable parameters trained once (like prompt tuning) that serve as task-conditioned anchors. The dynamic memory rides on top of this fixed scaffold.

Test-time training, properly framed

Titans is part of a broader 2024–2025 trend (Test-Time Training of Sun et al., DeltaNet, GLA, RWKV-7) that recasts state-space and linear-attention models as online learners of an associative key→value map. Titans is distinctive for: - Explicit momentum term (most contemporaries use SGD-only updates). - Per-dimension gates (vs. scalar). - Parameterizing the memory as an MLP rather than a linear matrix — a small nonlinear network can store more associations than $r^2$ scalars.

Benchmarks

  • Language modeling — competitive with or better than equivalently-sized Transformers and Mamba/Mamba2 on standard perplexity benchmarks.
  • Needle-in-a-haystack — scales to context >2M tokens, where Transformers blow up and SSMs degrade.
  • Commonsense reasoning — modest improvements over baselines.
  • Genomics, time series — wins on long-range dependency tasks where the relevant signal is hundreds of thousands of positions away.

Why it matters in the agent-memory line

  • First serious answer to “what would a learned memory module look like?” — earlier outside-channel methods (Memorizing Transformers, LongMem) used non-differentiable kNN; Titans makes the memory itself trainable, and updateable during inference.
  • Directly inspires δ-mem — δ-mem’s gated delta-rule update is structurally the same idea, simplified to a single matrix $S \in \mathbb{R}^{r \times r}$ instead of an MLP, and as a side-channel to a frozen Transformer rather than a fundamental architectural component.
  • Pre-trained vs. agent regime — Titans is trained from scratch alongside the backbone, so memory and attention co-adapt; δ-mem is for the agentic setting where a frozen pre-trained LLM is augmented post-hoc.

Open questions Titans does not fully resolve

  • Optimizer choice — why momentum-SGD specifically? Adam-style adaptive learning rates not explored.
  • Memory module capacity vs. width tradeoff — bigger MLP memory helps, but the scaling law isn’t characterized.
  • Compatibility with finetuned/agentic backbones — Titans is trained jointly; using a pre-trained backbone as the attention core (Frankenstein-style) isn’t tested.

Related notes
